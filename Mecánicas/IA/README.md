# 🧠 Sistema de IA para Videojuegos  
*Código y documentación para comportamientos de enemigos NPC (FSM, Pathfinding).*  

## **📦 Contenido**  
- `/scripts`: Código en C# (Unity) y GDScript (Godot).  
- `/docs`: Explicación técnica de los algoritmos usados.  
- `/ejemplos`: Escenas de prueba.  

## **🛠️ Cómo usar**  
1. Clona este repositorio.  
2. Copia la carpeta `scripts` a tu proyecto.  
3. Ajusta parámetros en `IAConfig.cs`.
